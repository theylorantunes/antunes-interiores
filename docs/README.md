# antunes-interiores
projeto pessoal dedicado a explorar minhas habilidades com tailwindcss

1. Introdução

    Nome do projeto: Antunes Interiores
    Objetivo: Criação de uma landingpage utilizando o framework TailwindCSS v3
    Público-alvo: Clientes interessados em design de interiores.


2. Tecnologias Utilizadas

    HTML5
    Tailwind CSS v3
    ScrollReveal.js
    Git para versionamento
    Mobile First como abordagem principal.


3. Estrutura do Projeto

    antunes-interiores/
    ├── index.html
    ├── assets/
    │   ├── img/
    │   └── css/
    ├── tailwind.config.js
    ├── package.json
    └── docs/
        └── README.md


4. Padrões de Versionamento

    SemVer: MAJOR.MINOR.PATCH

    Git Flow:

    main → Produção.
    develop → Desenvolvimento.
    feature/* → Novas funcionalidades.
    hotfix/* → Correções urgentes.




5. Mobile First

    Comece pelo layout para telas pequenas.
    Use breakpoints do Tailwind:

    sm: ≥ 640px
    md: ≥ 768px
    lg: ≥ 1024px
